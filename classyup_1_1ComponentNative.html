Detailed Description

Provides platform-native window and rendering capabilities for Components.

This class serves as an abstraction layer between the Component class and the platform-specific window management and rendering systems. It handles native window creation, event processing, and rendering pipeline setup across different platforms.

ComponentNative objects are typically created internally by the Component class when a Component needs to be displayed on screen as a top-level window.
