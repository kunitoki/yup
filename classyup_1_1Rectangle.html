Detailed Description

template<class ValueType>
class yup::Rectangle< ValueType >

Represents a rectangle in 2D space.

This class template provides a way to store and manipulate rectangles defined by their position (x, y) and size (width, height). The ValueType template parameter determines the type of these values (e.g., int, float). This class includes various methods for rectangle manipulations such as translation, scaling, intersection checks, and transformations among others.
