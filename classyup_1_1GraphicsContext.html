Detailed Description

Encapsulates a graphics context that abstracts rendering operations across various APIs.

This class serves as a base for implementing specific graphics context functionalities, such as rendering and resource management, across different graphics APIs like OpenGL, Direct3D, Metal, and Dawn. It offers a standardized interface for operations common to all graphics APIs.
