Detailed Description

template<typename Type, typename MutexType, bool onlyCreateOncePerRun>
struct yup::SingletonHolder< Type, MutexType, onlyCreateOncePerRun >

Used by the YUP_DECLARE_SINGLETON macros to manage a static pointer to a singleton instance.

You generally won't use this directly, but see the macros YUP_DECLARE_SINGLETON, YUP_DECLARE_SINGLETON_SINGLETHREADED, YUP_DECLARE_SINGLETON_SINGLETHREADED_MINIMAL, and YUP_IMPLEMENT_SINGLETON for how it is intended to be used.
