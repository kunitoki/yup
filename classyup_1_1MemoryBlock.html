Chops out a section of the block.

This will remove a section of the memory block and close the gap around it, shifting any subsequent data downwards and reducing the size of the block.

◆ toBase64Encoding()

String yup::MemoryBlock::toBase64Encoding ( ) const

Returns a string of characters in a YUP-specific text encoding that represents the binary contents of this block.

This uses a YUP-specific (i.e. not standard!) 64-bit encoding system to convert binary data into a string of ASCII characters for purposes like storage in XML. Note that this proprietary format is mainly kept here for backwards-compatibility, and you may prefer to use the Base64::toBase64() method if you want to use the standard base-64 encoding.

See also
fromBase64Encoding, Base64::toBase64, Base64::convertToBase64

◆ fromBase64Encoding()

bool yup::MemoryBlock::fromBase64Encoding ( StringRef encodedString)

Takes a string created by MemoryBlock::toBase64Encoding() and extracts the original data.

The string passed in must have been created by to64BitEncoding(), and this block will be resized to recreate the original data block.

Note that these methods use a YUP-specific (i.e. not standard!) 64-bit encoding system. You may prefer to use the Base64::convertFromBase64() method if you want to use the standard base-64 encoding.
