Detailed Description

template<typename Type>
struct yup::VariantConverter< Type >

This template-overloaded class can be used to convert between var and custom types.

If not specialised, the variant converter will attempt to use serialisation functions if they are detected for the given type. For details of what this entails, see the docs for SerialisationTraits.

In short, the constant 'marshallingVersion', and either the single function 'serialise()', or the function pair 'load()' and 'save()' must be defined for the type. These may be defined as public members of the type T itself, or as public members of yup::SerialisationTraits<T>, which is a specialisation of the SerialisationTraits template struct for the type T.
