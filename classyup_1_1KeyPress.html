Detailed Description

Represents a single key press, encapsulating information about the key and any associated modifiers.

This class is designed to represent a key press in a platform-independent manner, storing details about the key code, any modifier keys that were active (like Shift, Ctrl, etc.), and the Unicode character for key presses that generate text.
