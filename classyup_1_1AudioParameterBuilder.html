Detailed Description

A builder class for constructing AudioProcessorParameter instances using a fluent-style API.

This class simplifies the creation of AudioProcessorParameter objects by allowing you to configure each aspect step-by-step, including type, range, mapping functions, string conversion, and optional smoothing.

Example:

auto gainParam = AudioParameterBuilder{}
.withID ("gain")
.withName ("Gain")
.withRange (0.0f, 1.0f)
.withDefault (0.5f)
.withSmoothing (100.0f)
.build();
