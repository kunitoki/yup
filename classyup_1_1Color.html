Detailed Description

Represents an RGBA color for graphical use.

This class encapsulates color information in RGBA format, where each component (red, green, blue, alpha) is represented as an 8-bit value. It provides methods for adjusting color components individually, converting to and from HSL (Hue, Saturation, Luminance), and for performing operations like brightening, darkening, and contrasting.
